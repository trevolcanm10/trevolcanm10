## 🧠 Filosofía de Trabajo

* 🔍 **Comprensión Absoluta:** Analizo y desgloso la naturaleza y el dominio de un problema antes de escribir la primera línea de código.
* 🏗️ **Código Sostenible:** Priorizo la escritura de software limpio, modular, legible y con una arquitectura altamente mantenible a largo plazo.
* 📝 **Documentación como Pilar:** Considero que un desarrollo sin una excelente documentación técnica está incompleto.
* 🎨 **Sinergia Funcional:** Busco el balance ideal entre la lógica rigurosa del backend, el valor predictivo de los datos y la experiencia de usuario final.
